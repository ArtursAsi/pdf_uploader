### Instructions:
Clone project
>git clone https://github.com/ArtursAsi/pdf_uploader

Install Composer
>composer install

Create .env file
>cp .env.example .env

Link storage 
>php artisan Storage:link

Migrate Database
>php artisan migrate

Run app 
>php artisan serve


### Description

A simple PDF uploader. Upload a pdf file, it will be shown at the start.
